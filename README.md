# CS370
For this project, I was given starter code for the maze, helper classes, and parts of the neural network. My main task was completing the Q-training algorithm in the notebook. I built the training loop for the pirate agent using deep Q-learning. This included action selection, exploration vs. exploitation, storing experiences, training the model, and tracking win rates. I also added comments and improved performance.

Computer scientists solve problems by creating software, systems, and algorithms. Their work matters because technology is used in healthcare, business, education, and entertainment. In this project, I created an agent that learned from experience instead of following a fixed path. These same ideas apply to robotics, navigation, and automation.

I approach problems by understanding the goal and breaking it into smaller steps. In this project, I focused on movement decisions, rewards, and training results.
